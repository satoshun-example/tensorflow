# tensorflow example

## setup

```shell
$ brew install bazel swig
$ mkvirtualenv --python /usr/local/bin/python3.5 tensor
$ pip install -U six
$ pip install -U numpy
$ pip install wheel
$ pip install ipython
$ pip install --upgrade https://storage.googleapis.com/tensorflow/mac/tensorflow-0.6.0-py3-none-any.whl
```
